# مهام اليوم الثاني — التصميم والمعمارية

نسخة ويب تفاعلية وقابلة للتعبئة من كرّاسة «مهام اليوم الثاني» (Design Sprint · Day 2).
إعادة بناء وفيّة للأصل بصيغة PDF مع حقول إدخال تدعم العربية (RTL) وتتوسّع تلقائياً،
وحفظ تلقائي في المتصفح، وزر تصدير PDF بجودة الطباعة.

## المزايا
- 8 صفحات مطابقة للأصل (المحتوى، الألوان، الهندسة، الخطوط، الشعارات).
- حقول نصية تدعم العربية والإنجليزية وتنمو مع الكتابة دون قصّ.
- حفظ تلقائي محلي (localStorage) — لا تضيع مدخلاتك.
- زر **تصدير PDF / Export PDF** يطبع نسخة متجهية (A4، بدون هوامش، مع رسومات الخلفية).

## الاستخدام
افتح `index.html` عبر خادم محلي (أو الرابط المنشور)، عبّئ الحقول، ثم اضغط **تصدير PDF**.
في نافذة الطباعة اختر: الحجم **A4**، الهوامش **بلا**، وفعّل **رسومات الخلفية**.

A static, no-build site — deployable to GitHub Pages as-is.
